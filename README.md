# Predicting Employee Attrition using Classification Models

This repository contains all code and supporting materials for the individual classification project.

The goal of the project is to predict employee attrition using several machine learning classification techniques and to compare their performance across different models.

---

# Repository Contents

- `classification_code.R` – R script used for data preprocessing, model training (GLM, SVM, Neural Network), and evaluation  
- `employee_attrition.csv` – Dataset used in the analysis  
- `attrition_report` – Final report detailing methodology, results, and model comparison  
- `attrition_slides` – summary presentation
